# Voro-App

This is the reorganized application, ready for hosting.

## Structure
- `public/`: Contains all frontend HTML, CSS, and assets.
- `index.js`: Main backend server file.
- `database.js`: Database configuration.
- `college_data.db`: SQLite database file.

## How to Run
1. Open a terminal in this directory.
2. Run `npm install` to install dependencies.
3. Run `npm start` to start the server.
4. Open `http://localhost:4000` in your browser.

## Deployment
You can upload this entire `voro-app` folder to a hosting provider that supports Node.js (e.g., Render, Railway, Heroku).
