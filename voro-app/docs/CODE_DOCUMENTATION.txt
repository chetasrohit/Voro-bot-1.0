================================================================================
QUICK START GUIDE
================================================================================

The VoroAI system has 3 main parts:

1. SERVER (Backend) - Node.js + Express
   - index.js: Main server file
   - database.js: Database operations
   - user-auth.js: User login system
   - admin-api.js: Admin panel API

2. DATABASE - SQLite
   - college_data.db: Stores all data

3. FRONTEND (Client) - HTML + JavaScript
   - chatbot.html: Student chat interface
   - admin.html: Admin control panel

Read the detailed files for complete explanations.
